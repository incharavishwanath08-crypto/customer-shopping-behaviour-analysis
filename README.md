# Customer Shopping Behaviour Analysis – Python, SQL & Power BI

## 📌 Project Overview

This end-to-end data analytics project analyzes customer shopping behavior and demonstrates a complete analytics workflow using **Python, SQL, and Power BI**.

The project follows three stages:

1. **Python** – Data cleaning, preprocessing, feature engineering, and exploratory analysis.
2. **SQL** – Business analysis using analytical queries.
3. **Power BI** – Interactive dashboard creation for visual insights.

The objective is to transform raw customer shopping data into meaningful business insights and present them through an interactive dashboard.

---

## 📂 Project Workflow

### 1️⃣ Data Cleaning & Preprocessing (Python)

Performed using **Pandas** and **SQLAlchemy**.

#### Tasks completed:

* Imported and explored the dataset.
* Checked data types and descriptive statistics.
* Handled missing values in **Review Rating** using median values grouped by category.
* Standardized column names.
* Renamed columns for better readability.
* Created a new **Age Group** feature:

  * Young Adult
  * Adult
  * Middle-aged
  * Senior
* Converted purchase frequency into numerical days.
* Identified and removed redundant columns.
* Loaded the cleaned dataset into **MySQL** using SQLAlchemy.

#### Libraries Used

* Pandas
* NumPy
* SQLAlchemy
* PyMySQL

---

### 2️⃣ Business Analysis Using SQL

After importing the cleaned dataset into MySQL, various analytical queries were performed to answer business questions.

#### Key Analyses

* Revenue generated by male vs female customers.
* Customers using discounts but spending above average.
* Top-rated products based on review ratings.
* Purchase amount comparison between Standard and Express shipping.
* Spending behavior of subscribed vs non-subscribed customers.
* Products with the highest discount usage.
* Customer segmentation:

  * New Customers
  * Returning Customers
  * Loyal Customers
* Top 3 products within each category using window functions.
* Relationship between repeat buyers and subscription status.
* Revenue contribution by age group.

#### SQL Concepts Used

* Aggregate Functions
* GROUP BY
* ORDER BY
* Subqueries
* CASE Statements
* Window Functions
* Common Table Expressions (CTEs)
* Ranking Functions (ROW_NUMBER)
